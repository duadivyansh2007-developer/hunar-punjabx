# 🌾 HUNAR Punjab: AI-Powered Placement Ecosystem

**Bridging the Skill Gap in Punjab through Intelligent Career Matching.**

Welcome to the future of workforce management. **HUNAR Punjab** is a state-grade, AI-driven placement ecosystem designed to automate and optimize the entire career lifecycle for students, educational institutes, and recruitment partners across the Punjab region.

## 🔍 The Problem

The Punjab region faces a significant disconnect between higher education and industry requirements. Thousands of students graduate with skills that don't match the current job market's technical demands. **HUNAR Punjab** solves this by:
-   **Automating Skill Validation**: Using AI to parse resumes and predict job readiness.
-   **Bridging the Gap**: Directing students to institutional courses that specifically target local job market deficits.
-   **Centralizing Data**: A single platform for Students, Institutes, Companies, and Government Admin.

---

## 🛠️ The Tech Stack

-   **Frontend**: Next.js 14+ (App Router), Lucide Icons, Recharts (Analytics), Vanilla CSS (Glassmorphism & High-Premium Design).
-   **Backend**: Node.js, Express, MongoDB Atlas (Production-Ready Cloud Storage).
-   **AI Engine**: Simulated AI-scoring for candidate-job matching (Ready for full Python integration).
-   **Deployment**: Vercel (Production Cloud Hosting).

## 🚀 How to Run the Project Locally

### 1. Prerequisite
Ensure you have **Node.js 18+** and **Git** installed on your system.

### 2. Start the Backend
```bash
cd backend
npm install
# Copy .env.example to .env and add your variables
npm run dev
```

### 3. Start the Frontend
```bash
cd ../frontend
npm install
# Copy .env.example to .env and add your variables
npm run dev
```
Open your browser to `http://localhost:3000`.
